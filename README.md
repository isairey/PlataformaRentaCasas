# 🌌 Acerca del proyecto

**House Rental Management System** es un sistema desarrollado en Python orientado a la gestión eficiente de propiedades en renta, inquilinos y contratos habitacionales.

El proyecto fue diseñado para:

- 🏠 Gestionar viviendas
- 👥 Administrar inquilinos
- 📄 Controlar contratos
- 💰 Gestionar pagos y rentas
- 📅 Administrar disponibilidad
- 📊 Generar reportes
- 🔐 Implementar autenticación
- 🗄️ Gestionar información relacional

# ⚡ Instalación

## 📋 Requisitos

- Python 3.10+
- pip
- SQLite o MySQL
- Visual Studio Code
- Navegador moderno

---

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/HouseRentalManagementSystem.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd HouseRentalManagementSystem
```

---

## 3️⃣ Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Configurar base de datos

Crear base de datos:

```bash
house_rental_db
```

Importar:

```bash
database/house_rental.sql
```

---

## 5️⃣ Ejecutar proyecto

```bash
python main.py
```
